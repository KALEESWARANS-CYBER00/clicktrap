ClickTrap

Cybersecurity Awareness Project

Project Overview

ClickTrap is a cybersecurity awareness project designed to demonstrate how attackers can disguise files as normal images and educate users about safe file handling. When the user clicks the “fake” image, a fullscreen awareness message is displayed explaining the risks of opening unknown files.

Features

  • Fake image triggers awareness message.
  • Linux desktop integration via .desktop file.
  • Easy installation with installer script.
  • Cross-platform support (Linux + future Windows support).

Installation

Clone the repository and run the installer:

git clone https://github.com/KALEESWARANS-CYBER00/clicktrap.git
cd clicktrap
./install.sh

After running the installer, a “Holiday Photo” icon will appear on your Desktop. Clicking it will trigger the awareness demo.

Download

You can download ClickTrap directly from GitHub:

Download ZIP

Or clone the repository:

git clone https://github.com/username/clicktrap.git

Manual / User Guide

  1. Run install.sh to create the Desktop launcher.
  2. Click the “Holiday Photo” icon on the Desktop.
  3. The fake image will open and the awareness message will display.
  4. Close the message to finish the demo.
