ClickTrap

Cybersecurity Awareness Demonstration Tool

About

ClickTrap is a cybersecurity awareness project demonstrating how attackers can disguise files as harmless images. When a user clicks the fake image, an awareness screen appears explaining the risk of opening unknown files.

Features

  • Fake image trigger demonstration
  • Desktop launcher using Linux .desktop
  • Installer script for quick setup
  • Educational awareness message

Installation

git clone https://github.com/YOUR_USERNAME/clicktrap.git
cd clicktrap
chmod +x install.sh
./install.sh

After installation, a Holiday Photo icon appears on the Desktop.

Download

Download ZIP

Usage

  1. Run the installer.
  2. A Desktop icon called Holiday Photo appears.
  3. Click the icon.
  4. The disguised photo opens.
  5. A cybersecurity awareness message appears.
